# 前后端通信问题解决方案

## 📋 问题总结

### 发现的主要问题

1. **端点路径不匹配**
   - Android端调用 `/status`
   - 后端只有 `/api/v1/system/status`
   - ✅ 已修复: 添加了 `/status` 简单端点

2. **端口不一致**
   - Android端配置的端口: 8080
   - 后端实际端口: 8082
   - ⚠️  需要统一配置

3. **IP地址硬编码**
   - Android端硬编码了 `10.82.209.144`
   - 无法适应不同网络环境
   - ⚠️  需要动态配置

4. **缺少网络诊断工具**
   - 无法快速获取本机IP
   - 无法测试连接
   - ✅ 已创建诊断工具


### 方法2: 使用诊断工具自动更新 (推荐)

```bash

```

### 方法3: 动态配置 (最佳实践)

建议修改Android端代码，支持动态配置：

```kotlin
class ServerCommunicator(private val context: Context) {
    private val httpClient = OkHttpClient.Builder()
        .connectTimeout(10, TimeUnit.SECONDS)
        .readTimeout(30, TimeUnit.SECONDS)
        .writeTimeout(30, TimeUnit.SECONDS)
        .build()
    
    // 从SharedPreferences或配置文件读取
    private var serverUrl = getServerUrl()
    
    private fun getServerUrl(): String {
        val prefs = context.getSharedPreferences("server_config", Context.MODE_PRIVATE)
        return prefs.getString("server_url", "http://10.82.209.144:8080") ?: "http://10.82.209.144:8080"
    }
    
    fun setServerUrl(url: String) {
        serverUrl = url.trimEnd('/')
        // 保存到SharedPreferences
        val prefs = context.getSharedPreferences("server_config", Context.MODE_PRIVATE)
        prefs.edit().putString("server_url", serverUrl).apply()
    }
}
```

### 步骤3: 检查防火墙

**Windows防火墙设置:**

1. 打开 "Windows Defender 防火墙"
2. 点击 "高级设置"
3. 选择 "入站规则" → "新建规则"
4. 选择 "端口"
5. 选择 "TCP"，输入端口 "8082"
6. 选择 "允许连接"
7. 应用到所有配置文件

### 步骤4: 确认Android设备网络

在Android设备上：
1. 打开WiFi设置
2. 确认连接到与电脑相同的WiFi网络
3. 记下设备的IP地址

### 步骤5: 测试Android端连接

在Android设备上测试：

```kotlin
// 在Activity或Fragment中
```

## 🐛 常见错误及解决方案

### 错误1: Connection refused

**原因:** 后端未启动或端口被占用

**解决:**
```bash
# 检查端口是否被占用
netstat -ano | findstr :8082

# 如果被占用，停止占用该端口的程序
# 或者修改后端使用其他端口
```

### 错误2: Network unreachable

**原因:** Android设备与电脑不在同一网络

**解决:**
- 确保Android设备和电脑连接到相同的WiFi
- 检查网络设置

### 错误3: Timeout

**原因:** 防火墙阻止了连接

**解决:**
- 配置Windows防火墙允许8082端口
- 或者临时关闭防火墙测试

### 错误4: 404 Not Found

**原因:** 端点路径错误

**解决:**
- 确认使用 `/status` 端点
- 检查URL是否拼写正确
